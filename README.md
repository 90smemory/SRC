# InformationGather

SRC 资产信息聚合网站

## 介绍

SRC 资产信息聚合网站 收集了在进行资产收集时 常用的查询网站以及查询手段，方便进行SRC资产收集

本网站为静态网站，可直接部署在Git Page，Coding Page上

示例网站：https://90smemory.github.io/SRC/cn/index.html

网站效果图：

![demo](demo.png)

增删改查可修改assets/data.json文件
